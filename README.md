Ionic Photo Maps
=====================

Application example for describing native plugins.

## Using this project

You'll need the Ionic CLI with support for v2 apps:

```bash
$ npm install -g ionic
```

Then run:

```bash
$ ionic run android
```
or
```bash
$ ionic run ios
```
